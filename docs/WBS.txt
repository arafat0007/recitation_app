# 1. Project Setup and Technical Foundation

## 1.1 Create project

### 1.1.1 Initialize Android project

* Create new Android project in Android Studio
* Choose Kotlin
* Enable Jetpack Compose
* Set minimum SDK
* Set package name
* Configure portrait-only behavior

### 1.1.2 Configure dependencies

* Add Compose dependencies
* Add Navigation Compose
* Add Material 3
* Add Media3 / ExoPlayer
* Add Room
* Add DataStore
* Add Firebase / Firestore
* Add lifecycle / ViewModel / coroutines
* Add YouTube player approach for Owaj
* Add image loading library if thumbnails are used

### 1.1.3 Configure architecture base

* Create package structure:

  * `app`
  * `core`
  * `data`
  * `domain`
  * `feature_home`
  * `feature_surah`
  * `feature_doa`
  * `feature_owaj`
  * `feature_favorites`
  * `feature_settings`
* Decide architecture pattern:

  * MVVM
  * Repository pattern
  * Use cases optional for MVP

### 1.1.4 Configure Firebase project

* Create Firebase project
* Connect app to Firebase
* Enable Firestore
* Prepare Firestore collections structure
* Set basic security rules for creator-controlled content

### 1.1.5 Version control and environment

* Create Git repository
* Create `.gitignore`
* Set branch strategy
* Add README with project purpose and setup steps

A running blank Android app with all main dependencies connected.
